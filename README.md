### Code Challenge

For this test, I created a simple calculator.

### Installation

Just clone the repo with

    git clone https://github.com/eccianime/simple_calculator.git

and install with

    npm install

or

    yarn

### Running the app

Go to the folder with

    cd simple_calculator

and run

    npm run start

or

    yarn start
